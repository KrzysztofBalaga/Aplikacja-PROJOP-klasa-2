Podstawowe Komendy Javy


Oto prosty kod w języku java:



Metoda "main" jest publiczna, statyczna i zwraca wartość typu void (czyli nie zwraca żadnej wartości). Przyjmuje ona argument typu String[], który reprezentuje listę argumentów przekazywanych do programu podczas jego uruchamiania.

Wewnątrz metody "main" znajduje się tylko jedna instrukcja, która wyświetla tekst "Hello World" na standardowym wyjściu przy użyciu metody "println" klasy "System". Metoda "println" wyświetla tekst wraz z przejściem do nowej linii.

Podsumowując, kod ten jest przykładem najprostszego programu napisanego w języku Java, który wyświetla jedynie napis "Hello World" na ekranie.


Rodzaje Zmiennych



Oto najprostrze rodzaje zmiennych:


1. Zmienna "a" jest typu całkowitoliczbowego (int) i została zainicjowana wartością 1. Typ int reprezentuje liczby całkowite (bez ułamków) o zakresie od -2^31 do 2^31-1.

2. Zmienna "b" jest typu zmiennoprzecinkowego (double) i została zainicjowana wartością 1.5 (warto zaznaczyć, że liczby zmiennoprzecinowe posiadają kropke a nie przecinek). Typ double reprezentuje liczby zmiennoprzecinkowe (z ułamkami) o podwójnej precyzji.

3. Zmienna "c" jest typu znakowego (char) i została zainicjowana wartością 'a'. Typ char reprezentuje pojedynczy znak Unicode. W Javie znak musi być umieszczony w apostrofach ('a'), a nie w cudzysłowiach("a").

4.Zmienna "v" jest typu logicznego (boolean) i została zainicjowana wartością true. Typ boolean reprezentuje wartość logiczną true (prawda) lub false (fałsz).

5.Zmienna "w" jest typu łańcuchowego (String) i została zainicjowana wartością "abc123". Typ String reprezentuje ciąg znaków i jest używany do przechowywania tekstu lub łańcuchów znaków.


Operacje na Zmiennych



Oto podstawowe operatory zmiennych:


W pierwszej linijce obliczeń widać "System.out.println("a + b = " + (a+b));", zapis ten oznacza, że zostanie wydrukowany tekst "a + b = " oraz wynik dodawania a+b, czyli "a + b = 12" (tak samo z pozostałymi przykładami).

1. Dodawanie (+) i odejmowanie (-) dla zmiennych typu int, double, char i String. Wyniki będą wyświetlane jako ciągi znaków z wynikiem obliczeń.

-Wynik dodawania "a + b" to 12.
-Wynik odejmowania "a - b" to -4.
-Wynik dodawania "x + y" to 5.8.
-Wynik odejmowania "x - y" to -2.8.
-Wynik dodawania "c + d" to 206 (sumowanie wartości numerycznych dla kodów Unicode znaków 'a' i 'm').
-Wynik dodawania "Natan Jest mily" (łączenie ciągów znaków "Natan", "Jest", "*spacja*" i "mily").
-Wynik dodawania "Natan mily Jest" (łączenie ciągów znaków "Natan", "mily", "*spacja*" i "Jest").

2. Mnożenie (*) i dzielenie (/) dla zmiennych typu int i double. Wyniki będą wyświetlane jako ciągi znaków z wynikiem obliczeń.

-Wynik mnożenia "a * b" to 32.
-Wynik dzielenia "a / b" to 0, ponieważ oba operandy są typu int, a wynik dzielenia również jest typu int, co oznacza, że liczba zostanie zaokrąglona w dół do liczby całkowitej.
-Wynik mnożenia "x * y" to 6.45.
